SEARCH RESULTS

Limit the search using the controls.

To bring up a search preview, click on a search category and select a result in text form, or click on an object icon. In a search preview, you can select matching objects from a matching copy, and you can change copies.

Click 'G' in the banner to enter Gallery Mode. In Gallery Mode, objects will open by default in the Object View.

Click 'R' in the banner to enter Reading Mode. In Reading Mode, objects will open by default in the Reading View.
