SEARCH RESULTS

Limit the search using the controls.

To bring up a search preview, click on a search category, then select a result as text, or click on an object icon. In a search preview, you can select matching objects from a matching copy, and you can change copies.

BANNER

  • Click on the navicon to browse Blake's works.
  • Click on 'G' to enter Gallery Mode. In Gallery Mode, objects will open by default in the Object View, which shows them in a gallery window accompanied by their critical apparatus.
  • Click on 'R' to enter Reading Mode. In Reading Mode, objects will open by default in the Reading View, which shows them accompanied by their transcriptions in a horizontally scrollable window and excludes a critical apparatus.
